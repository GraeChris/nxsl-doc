=== TableColumn

Represents table column definition object (used by Table class).

`dataType => Integer`::
<<const-dci-datatype,Data type>>

`displayName => String`::
Display name

`isInstanceColumn => Boolean`::
`TRUE` if column is marked as instance column

`name => String`::
Column name
