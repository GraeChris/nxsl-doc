== AgentReadParameter(`node`, `name`)

Request `name` metric directly from agent on given <<class-node, node>> and return it's value or `null` if failed.

.Example
----
>>> v = AgentReadParameter($node, "Agent.Version")
>>> println(v)
2.0.2
----
