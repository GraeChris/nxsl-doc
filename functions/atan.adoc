[[func-atan]]
== atan()

[source,c]
----
atan(number) => Number
----

Calculates arc tangent x

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|number|Number|Real number x, with −1 ≤ x ≤ 1
|===

.Return

The angle in radians whose arc tangent is x

.Example
[source,c]
----
atan(1) //Result: 0.785398
atan(0.5) //Result: 0.463648
----
