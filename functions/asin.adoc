[[func-asin]]
== asin()

[source,c]
----
asin(number) => Number
----

Calculates arc sine x

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|number|Number|Real number x, with −1 ≤ x ≤ 1
|===

.Return

The angle in radians whose sine is x

.Example
[source,c]
----
asin(1) //Result: 1.570796
asin(0.5) //Result: 0.523599
----
