[[func-gethostbyname]]
== gethostbyname()

[source,c]
----
gethostbyname(hostname) => String
----

Resolve hostname to IP address

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|hostname|String|Hostname 
|===

.Return
IP address as a String

.Example
[.source]
....
println gethostbyname("localhost"); //Will print "127.0.0.1"
....
