[[func-gethostbyaddr]]
== gethostbyaddr()

[source,c]
----
gethostbyaddr(address) => String
----

Resolve IP address to host name

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|address|String|IP address to resolve host name
|===

.Return
Host name

.Example
[.source]
....
println gethostbyaddr("127.0.0.1"); //Will print "localhost"
....
