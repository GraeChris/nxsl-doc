[[func-tan]]
== tan()

[source,c]
----
tan(x) => Number
----

Calculates tangent x

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Number|Angle in radian 
|===

.Return
Result of tangent for this angle

.Example
[source,c]
----
print tan(0.5); //will print 0.546302
----
