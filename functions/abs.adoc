[[func-abs]]
== abs()

[source,c]
----
abs(number) => Number
----

Returns the absolute value of the number.

.Parameters
[cols="1,3" grid="none", frame="none"]
|===
|number|Input value.
|===

.Return

Absolue value of the input.

.Example
[.source]
....
>>> abs(12.3)
12.3
>>> abs(-0.307)
0.307
....
