[.nxsl-function]
[[func-lower]]
== lower()

[source,c]
----
lower(string) => String
----

Converts string to lowercase string.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|string|String|String to convert
|===

.Return
Source string converted to lowercase.

.Example
[.source]
....
println(lower("aBcD")); // Will print "abcd"
....
