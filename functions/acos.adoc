[[func-acos]]
== acos()

`acos(number) => Number`::

Calculates arc cosine x

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|number|Number|Real number x, with −1 ≤ x ≤ 1
|===

.Return

The angle in radians whose cosine is x

.Example
[source,c]
----
acos(-1) //Result: 3.141593
acos(0.5) //Result: 1.047198
----
