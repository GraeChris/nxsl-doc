== exp()

[source,c]
----
exp(input) => Float
----

Computes `e**x`, the http://en.wikipedia.org/wiki/Exponential_function[base-e] exponential.

.Parameters
[cols="1,3" grid="none", frame="none"]
|===
|input|Input number.
|===

.Return

.Example
[.output]
....
>>> exp(2)
7.3890561
....
