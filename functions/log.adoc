[[func-log]]
== log()

[source,c]
----
log(x) => Number
----

Calculates natural logarithm

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Number|Number to calculate natural logarithm of
|===

.Return
Natural logarithm of x.

.Example
[.source]
....
ptintln log(2); // Will print "0.693147"
....
