== CurrencyExponent()

[source,c]
----
CurrencyExponent(code) => Integer
----

Get currency exponent

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|code|String|Currency numeric or character code as a String 
|===

.Return
Currency exponent

.Example
[.source]
....

println CurrencyExponent("978"); //Will print 2
