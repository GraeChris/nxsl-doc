[[func-sinh]]
== sinh()

[source,c]
----
sinh(x) => Number
----

Calculates hyperbolic sine x

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Number|Angle in radian 
|===

.Return
Result of hyperbolic sine for this angle

.Example
[source,c]
----
print(sinh(0.5)); //will print 0.521095
----
