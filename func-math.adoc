=== Math

==== abs(`number`)

Returns the absolute value of `number`.

  >>> abs(12.3)
  12.3
  >>> abs(-0.307)
  0.307

==== ceil(`number`)

Round up `number`.

  >>> ceil(2.3)
  3.0
  >>> ceil(3.8)
  4.0
  >>> ceil(-2.3)
  -2.0
  >>> ceil(-3.8)
  -3.0

==== exp(`x`)

Computes `e**x`, the http://en.wikipedia.org/wiki/Exponential_function[base-e] exponential of `x`

  >>> exp(2)
  7.3890561

==== floor(`x`)

Return the largest integral value <= `x`.

  >>> floor(2.3)
  2
  >>> floor(3.8)
  3
  >>> floor(-2.3)
  -3
  >>> floor(-3.8)
  -4
